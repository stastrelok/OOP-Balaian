# Додаткове завдання: Узагальнений кеш (Cache<T>)

## Опис завдання
Реалізувати узагальнений клас кешу з використанням обмежень типів (Generic Constraints), власним механізмом очищення пам'яті та ручним алгоритмом сортування.

## Технічні деталі
1. **Обмеження (Constraints):** Використано `where T : class, new()`. Це гарантує, що кеш працюватиме лише з об'єктами класів, які можна створити через `new()`.
2. **Алгоритм видалення:** Реалізовано стратегію **FIFO (First In, First Out)**. При спробі додати (N+1)-й елемент, найперший доданий елемент автоматично видаляється.
3. **Алгоритм сортування:** Використано **Bubble Sort (Сортування бульбашкою)** для впорядкування елементів за міткою часу (`DateTime`) без використання вбудованих методів LINQ.
